`x^2 + 6x - 7 = 0`

`x^2 + 6x = 7 `
`x^2 + 6x + \(6/2\)^2 = 7 + 3^2`
`(x+3)^2 = 16`
`x + 3 = +- sqrt(16)`
`x = +-4 - 3`
`x=1, x=-7`

`x^2 + 2x - 8 = 0`

`x^2 + 2x = 8`
`x^2 + 2x + \(2/2\)^2 = 8 + 1`
`\(x+1\)^2 = 9`
`x+1 = +- sqrt(9)`
`x=+-3-1`
`x=2, x=-4`

`x^2 - 4x + 3 = 0`

`x^2-4x = -3`
`x^2-4x + \(-4/2\)^2 = -3 + \(-2\)^2`
`\(x-2\)^2 = 1`
`x-2 = +- sqrt(1)`
`x=+-1+2`
`x = 3, x = 1`

`x^2 - 8x - 9 = 0`

`x^2-8x =9`
`x^2-8x+\(-8/2\)^2 = 9+\(-4\)^2`
`\(x-4\)^2 = 25`
`x-4 = +- sqrt(25)`
`x = +-5 + 4`
`x=9, x=-1`
