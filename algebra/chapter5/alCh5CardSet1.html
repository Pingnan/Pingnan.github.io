Distance & Midpoint

Find the distance and midpoint of each pair of points

(Click on a card to see the solution)


`\(1, 2\)` and `\(4, 5\)`

`d = sqrt(\(4-1\)^2 + \(5-2\)^2)`
`= sqrt(9+9) = sqrt(18)`

`x_M = (1+4)/2 = 2.5`
`y_M = (2+5)/2 = 3.5`

`\(-3, 2\)` and `\(4, -3\)`

`d = sqrt(\(4-\(-3\)\)^2 + \(-3 - 2\)^2)`
` = sqrt(49+25) = sqrt(74)`

`x_M = (-3 + 4)/2 = 0.5`
`y_M = (2 + \(-3\))/2 = -0.5`

`\(-1, -4\)` and `\(5, 7\)`

`d = sqrt(\(5 - \(-1\)\)^2 + \(7 - \(-4\)^2)`
`= sqrt(36 + 121) = sqrt(157)`

`x_M = (-1 + 5)/2 = 2`
`y_M = (-4 + 7)/2 = 1.5`

`\(3, -5\)` and `\(-8, 6\)`

`d = sqrt(\(-8 - 3\)^2 + \(6 - \(-5\)\)^2`
`= sqrt(121+121) = sqrt(242)`

`x_M = (3 + \(-8\))/2 = -2.5`
`y_M = (-5 + 6)/2 = 0.5`
